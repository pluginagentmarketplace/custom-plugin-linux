# /learn - Create Learning Path

Start a personalized learning journey for any specialization.

## Usage: `/learn [role] --level [beginner|intermediate|advanced] --pace [slow|moderate|fast]`

## Examples
- `/learn frontend --level beginner --pace moderate`
- `/learn machine-learning --level intermediate --pace fast`
- `/learn devops --level advanced`

## Output
- Step-by-step learning path
- Estimated time (hours/weeks)
- Resources & materials
- Projects & milestones
- Assessment checkpoints
