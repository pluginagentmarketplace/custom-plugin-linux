# /compare - Compare Specializations

Compare two or more roles side-by-side to help choose your path.

## Usage: `/compare [role1] [role2] [role3...]`

## Examples
- `/compare frontend backend`
- `/compare python java go`
- `/compare product-manager engineering-manager`
- `/compare react vue angular`

## Comparison Includes
- Required skills
- Salary ranges
- Job market demand
- Learning difficulty
- Time to proficiency
- Career progression
- Similar roles
